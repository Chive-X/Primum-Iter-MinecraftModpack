# 《初程》Minecraft整合包
本整合包集成了一些优化模组和许多对原版内容的调整，尽量避免添加新物品，旨在作为更复杂整合包的基础，也可以直接作为风味原版游玩。总体而言，难度略低于原版，舒适度和性能高于原版。
*「初程」，意为「最初的旅程」。无论是机械还是魔法，战斗还是建筑，总归得从这个无比熟悉的地方开始。*
## 介绍
### 模组筛选原则
- 必要：
- 实用：
- 轻量：所选的模组应当降低性能消耗，尽量不添加新物品。
- 兼容：为了兼容复杂整合包中的模组，避免过于激进的优化
本整合包的风格虽然紧贴原版，但包含许多游戏机制变动，因此无法直接进入原版服务器。原版生电的兼容性未测试，不保证所有特性都能使用。
不包含大小地图和领土模组，不包含修改玩家动画和摄像头的模组，不包含原版结构改良模组。不包含Fancymenu、FTBQuest等魔改模组。不包含备份模组。
包含信雅互联，不过所装载的Fabric模组均为轻量的客户端改进。
支持光影
### 版本支持
本整合包暂无多版本开发的计划，仅支持目前的模组主流版本1.20.1。拟支持Forge和Fabric两端，但Forge优先，且部分模组会有出入。
推荐使用Java 17启动。

#### 本地化支持
本整合包完全支持的语言有zh_cn（简体中文、zh_tw（台湾繁体中文）、zh_hk（香港繁体中文）、en_us（美式英文）、lzh（文言文）。其他语言暂未支持，但对原版游戏来说，Minecraft自身的本地化支持就基本够用了。本地化支持主要集中在配置界面、工具提示等处。个别硬编码内容未支持。
