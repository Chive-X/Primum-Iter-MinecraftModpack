# 《初程》Minecraft整合包（zh_cn）
本整合包集成了一些优化模组和许多对原版内容的调整，尽量避免添加新物品，旨在作为更复杂整合包的基础，也可以直接作为风味原版游玩。
*「初程」，意为「最初的旅程」。无论是机械还是魔法，战斗还是建筑，总归得从这个无比熟悉的地方开始。*
## 介绍
### 模组筛选原则
- 必要：
- 实用：
- 轻量：所选的模组应当降低性能消耗，尽量不添加新物品。
- 兼容：为了兼容复杂整合包中的模组，避免过于激进的优化
本整合包的风格虽然紧贴原版，但包含许多游戏机制变动，因此无法直接进入原版服务器。原版生电的兼容性未测试，不保证所有特性都能使用。
不包含大小地图模组，不包含玩家动画、摄像头相关模组。
包含信雅互联，不过所装载的Fabric模组均为轻量的客户端改进。
支持光影
### 版本支持
本整合包暂无多版本开发的计划，仅支持目前的模组主流版本1.20.1。拟支持Forge和Fabric两端，但Forge优先，且部分模组会有出入。
推荐使用Java 17启动。
